This folder contains data to accompany v55i12.R.

There are four files.

1)  DLBCL.txt - the DLBCL data

2)  DLBCL.clusters.txt - the cluster labels given by manual gating for DLBCL

3)  plot.parameters - graphical parameters used to generate Figure 2 in the manuscript


References for the data:
Spidlen J, Breuer K, Rosenberg C, Kotecha N and Brinkman RR. 
    FlowRepository - A Resource of Annotated Flow Cytometry Datasets Associated with Peer-reviewed Publications. 
	Cytometry A. 2012 Sep; 81(9):727-31.

Aghaeepour N, Finak G; FlowCAP Consortium; DREAM Consortium, Hoos H, Mosmann TR, Brinkman R, Gottardo R, Scheuermann RH.
    Critical assessment of automated flow cytometry data analysis techniques.
	Nature Methods. 2013 Mar; 10(3):228-38.
